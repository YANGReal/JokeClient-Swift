JokeClient-Swift
================
1.基于swift

2.用糗事百科的API简单做了一个糗百客户端,可以浏览各种段子,浏览搞笑图片,查看精彩评论等.

3.本项目仅供学习交流使用,不能用作任何形式的商业用途!内容接口来源于网络上公开的API,如侵犯到相关机构的权益,本人不承担任何后果

4.修复一些编译错误
